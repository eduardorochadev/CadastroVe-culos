# Projeto Veículos API
>  Projeto para teste prático

## Requirements
- Node versão 1.2.3
- Mocha
  - Xyz
     
## Setup


npm install


npm start


## Test


npm run test


## Endpoints

- POST - Adicionar um novo veículo
- GET - Listar todos os veículos
- GET - Listar um veículo por ID
- PUT - Atualizar um veículo por ID
- DELETE - Deletar um veículo por ID
